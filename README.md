# Skill Exchange Platform

## Graduation Project

Skill Exchange Platform is a web-based application that enables users to share, learn, and exchange skills through an interactive community.

## Features

* User Registration and Login
* Profile Management
* Skill Posting
* Skill Search
* Messaging System
* Ratings and Reviews
* Admin Dashboard

## Technologies

* Python
* Flask
* PostgreSQL
* HTML5
* CSS3
* JavaScript
* Bootstrap

## Team Role

* Team Leader
* System Analysis
* Database Design
* Backend Development

## Project Objective

The main objective of this project is to create an online platform that connects users who want to learn new skills with users who can teach or share their expertise, providing a collaborative and interactive learning environment.
